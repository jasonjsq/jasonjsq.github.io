git

Git 的安装
git init ->生成一个.git隐藏文件
设置git 的用户名邮箱
git config –global user.name/.email只需要设置一次
Git 分本地仓库和远端仓库（Github）

本地仓库 工作区->暂存区->Git仓库
git add（文件名）（文件名1 文件名2）（.）
添加到暂存区
git commit + -m ****
暂存区到本地仓库
Git操作版本回退（时光机操作）
查看版本 git log //git log–pretty==oneline 只有一行且有注释
回退指令 git reset–hard 提交编号
回到历史指令 git reflog

远端仓库
Github 注册
新建库->两种传输方式SSH/HTTP
SSH传输需要配置秘钥
ssh-keygen -t -rsa -C “账号”->回车三连
到user文件夹下.ssh文件下存放公钥私钥
到GitHub上进行搭建SSH
HTTP
第一次传输时需要修改配置本地文件（设置GitHub的账号密码）
.git/config 修改url的github前加账号：密码
克隆clone
git clone 网址
发送push
git push
拉取pull
git pull

分支管理
master主分支
查看分支 git branch
创建分支 git branch 分支名
切换分支 git checkout 分支名
删除分支 git branch -d 分支名
合并分支 git merge 被合并的
